# /rambling - Write down current analysis

Write down the current analysis, discussion, or insight in a document for future reuse.

Though not necessarily useful to build skills, or plans, ramblings store acquired knowledge for the future.

## What to Extract

Anything we have been discussing, exchanging, brainstorming about in the latest session.

## Output Format

Create a complete structured document about the ongoing conversion in a file at `.claude/plans/ramblings/[pattern-name].md`.
